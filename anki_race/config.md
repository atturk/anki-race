# Anki Race Configuration Guide

Here is a summary of the configuration keys available for Anki Race:

- **road_scrolling**: `true` if the road texture should scroll infinitely, `false` otherwise.
- **show_overview_button**: `true` to display the "Race" button on the deck overview page.
- **default_mode**: `"normale"` for the Standard Race mode, `"fuga"` for the Pursuit Escape mode.
- **default_time**: The default CPU duration limit in minutes (e.g., `5.0`).
- **default_advantage**: The default user starting position advantage percentage in Escape mode (e.g., `30.0`).
- **road_height**: The height of the race track in pixels (default is `35`).
- **car_cpu_size**: The width and height of the opponent's vehicle in pixels (default is `32`).
- **car_user_size**: The width and height of your vehicle in pixels (default is `32`).
- **nitro_enabled**: `true` to enable Nitro Boost streak event, `false` to disable.
- **nitro_cards**: The number of consecutive Good/Easy answers needed to trigger the Nitro Boost (default is `5`).
- **decor_enabled**: `true` to enable road decorations, `false` to disable.
- **decor_type**: `"emoji"` or `"image"`.
- **decor_emoji**: The emoji or text to replicate as road decoration.
- **decor_spacer**: The spacing in pixels between decoration elements.
- **decor_scrolling**: `true` to scroll decorations infinitely, `false` otherwise.
- **decor_speed**: The speed of the decoration scrolling animation.
